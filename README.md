# Sails-Angular

This is an example Sails.js and AngularJs aplication, it was built with Sails.js v0.11.0, AngularJS v1.4 and Bootstrap v3.3.4 SASS

## How to use it

To use it just run:

    git clone https://github.com/danymar24/sails-angular.git
    cd sails-angular
    npm install
    sails lift
